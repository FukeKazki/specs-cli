# R-002 データ構造

ユビキタス言語は `domain/glossary/<term>.md` (type: term)、
モデルは `domain/models/<model>.md` (type: model) として 1 件 1 ファイルで管理する。
名前に日本語を許可し、パス区切り (`/` `\`) や `.` 始まりの名前は禁止する。
