# Screen: かんばんビュー画面

## 目的

status を列としたボードで進捗を可視化し、ドラッグ&ドロップで状態を遷移させる画面。

## 項目

- 列見出し（status: 未着手 / 進行中 / レビュー / 完了）
- カード: 1 要件（要件名 / priority / feature / assignee / due）

## 操作

- [x] A-001 カードを別の列へドラッグ&ドロップして status を更新・保存する
- [x] A-002 feature・priority・担当でのフィルタ
- [x] A-003 カードを選択して詳細画面へ遷移する

## エラー

- status 更新（保存）失敗時はトーストを表示し、カードを元の列へ戻す
- 要件が 0 件の場合は空状態を表示する
