# Screen: 仕様書詳細画面

## Purpose

選択した仕様書の内容を閲覧・編集する画面。

## Fields

- パンくず（features / feature名 / ファイル名）
- frontmatter（メタ情報枠）
- 本文（Markdown整形表示 / 編集時はテキストエリア）

## Actions

- 編集 → 保存 / キャンセル
- 削除（確認ダイアログ後に実行）

## Errors

- 取得・保存・削除の失敗時はトーストでメッセージを表示する
- 存在しない仕様書を開いた場合はエラーメッセージを表示する
