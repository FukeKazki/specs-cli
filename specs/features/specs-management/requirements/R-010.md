# R-010 MoSCoW

各要件に MoSCoW 優先度（Must / Should / Could / Won't）を付けられる。

要件は frontmatter の `priority` に優先度を持つ（`Must` / `Should` / `Could` / `Won't`）。
リスト / かんばん / ガントの各ビューでは色分けタグとして描画する
（Must=赤 / Should=橙 / Could=青 / Won't=灰）。priority の無い要件はタグ無し。
