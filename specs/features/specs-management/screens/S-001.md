# Screen: 仕様書一覧画面

## 目的

全仕様書を俯瞰し、目的の仕様書へ遷移する起点となる画面。

## 項目

- feature 名（グループ見出し）
- ファイル名（spec.md / R-00n.md）
- 画面名（screens の各項目）
- type バッジ（feature / api / screen）

## 操作

- [x] A-001 仕様書を選択して詳細画面へ遷移する
- [x] A-002 「新規 feature」から feature を作成する
- [x] A-003 「+ 画面」から画面を作成する
- [x] A-004 画面をドラッグ&ドロップで並び替える

## エラー

- 一覧取得失敗時はトーストでメッセージを表示する
- 仕様書が0件の場合は「仕様書がありません」を表示する
