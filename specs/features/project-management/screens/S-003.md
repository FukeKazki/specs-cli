# Screen: ガントチャート画面

## 目的

start〜due を期間とする横棒で、要件の時系列と依存関係を俯瞰する画面。

## 項目

- 時間軸（日 / 週 / 月）
- 要件バー（start〜due、status で色分け）
- 依存線（depends_on）
- 未スケジュール枠（start / due 未設定の要件）

## 操作

- [ ] A-001 バーのドラッグ / リサイズで start・due を更新・保存する
- [x] A-002 feature・status・担当でのフィルタとグルーピング
- [x] A-003 バーを選択して詳細画面へ遷移する

## エラー

- 保存失敗時はトーストを表示し、バーを元の位置へ戻す
- start > due など不正な期間は保存せずエラーを表示する
