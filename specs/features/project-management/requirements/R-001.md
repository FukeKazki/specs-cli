# R-001 進捗管理の単位（要件）

進捗管理の単位は要件（Requirement）とする。
`features/<feature>/requirements/R-00n.md` 1 件を進捗管理の 1 項目として扱い、
feature 単位でまとめる。要件は specs-management が管理する仕様ファイルの一種で、
frontmatter に `priority`（MoSCoW）・`status`・`assignee`・`start`・`due`・`depends_on` を持つ。
個別タスクの分解・消化は対象外で、本機能は要件の進行状況を扱う。
進捗メタ情報（status / 担当 / 期間など）は対象ファイルの frontmatter を唯一の正とする。
