# R-010 データの永続化と安全性

進捗データはすべて各要件ファイルの frontmatter に保存し、DB を用いない
（Markdown を唯一の正とする）。
ID（specs/ からの相対パス）はパストラバーサルを禁止し、管理対象ルート配下かつ
`.md` / `.yaml` で終わるパスのみを受け付ける。
