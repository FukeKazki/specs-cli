# R-003 スケジュール属性

ガント表示のため、要件ファイルの frontmatter に以下の任意項目を持てる。未設定でもよい。

- `assignee`: 担当者（文字列）
- `start`: 開始日（`YYYY-MM-DD`）
- `due`: 期限日（`YYYY-MM-DD`）
- `depends_on`: 依存する要件の id（相対パス）の配列
