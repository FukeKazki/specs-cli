# Screen: 仕様書詳細画面

## 目的

選択した仕様書の内容を閲覧・編集する画面。

## 項目

- パンくず（features / feature名 / ファイル名）
- frontmatter（メタ情報枠）
- 本文（Markdown整形表示 / 編集時はテキストエリア）
- サイドバー 

## 操作

- [x] A-001 編集 → 保存 / キャンセル
- [x] A-002 削除（確認ダイアログ後に実行）
- [x] A-003 サイドバーでstatusを変更できる
- [x] A-004 Actionの追加、Actionsの近くに＋ボタンを用意し押すとActionsに新しいリストとIDが採番された状態で編集画面に遷移する

## エラー

- 取得・保存・削除の失敗時はトーストでメッセージを表示する
- 存在しない仕様書を開いた場合はエラーメッセージを表示する
