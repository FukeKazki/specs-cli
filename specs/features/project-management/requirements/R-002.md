# R-002 ステータス

frontmatter の `status` を進捗ステータスとして扱う。標準値は次の 4 つとし、
かんばんの列・リストの状態・ガントの色分けに共通で用いる。

- `draft`（未着手） / `in-progress`（進行中） / `review`（レビュー） / `done`（完了）

未知の値はそのまま 1 つの状態として扱い、消失させない。
