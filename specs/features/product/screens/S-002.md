# Screen: プリンシパル画面

## Purpose

`product/principles.md` の閲覧・編集ができる。

## Fields

- Principle N（判断基準となる原則）

## Actions

- 本文を Markdown として整形表示する
- 「編集」で編集モードへ切り替える
- 「保存」で本文を上書き保存する / 「キャンセル」で破棄する

## Errors

- 取得失敗時はトーストでメッセージを表示する
- 保存失敗時はトーストを表示し編集モードを維持する
