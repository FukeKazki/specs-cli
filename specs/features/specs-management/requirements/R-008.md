# R-008 画面の並び替え

一覧上のドラッグ&ドロップで同一 feature 内の画面の表示順を変更できる。
並び順は各画面ファイルの frontmatter の `order` に永続化する。
一覧は domain → feature 昇順 → feature 内は spec.md, requirement, screen を
order 昇順で表示する。
