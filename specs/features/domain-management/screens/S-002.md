# Screen: モデル一覧画面

## Purpose

モデルが一覧で表示される

## Fields

- モデル名（一覧）
- Description（詳細）
- mermaid 図（詳細・preview 描画）

## Actions

- モデルの追加（「+ モデル」）
- モデルの編集（詳細画面で編集 → 保存）
- モデルの削除
